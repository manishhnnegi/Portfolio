GitHub Code Post

  • Gradient Calculation in Neural Network with Karpathy's Micrograd: In this Code post, we’ll explore how gradients are calculated in neural networks, based on Karpathy’s insightful lecture and his Micrograd library. Gradients play a crucial role in optimizing the performance of neural networks, and understanding how they work is key to grasping backpropagation and network training.
  • Bigram character level Language modeling: This code post explores the concept of Bigram Character-Level Language Modeling, which focuses on predicting the next character in a sequence based on the previous character. By utilizing bigrams, we can capture local dependencies in text data, making it a powerful approach for tasks such as text generation and completion.
  • Character Level Language Modeling Using MLP (Multi-Layer Perceptron): This code post presents a character-level language modeling approach utilizing a Multi-Layer Perceptron (MLP) to predict the next character in a sequence based on a given input. The model is trained on a dataset containing approximately 32,033 different names in the English language, enabling it to learn the patterns and structures inherent in character sequences.
  • Language Translation with Transformer: This Code post focuses on developing a language translator that converts English text into Italian using an attention-based Transformer architecture. The goal is to implement a robust translation model capable of understanding and generating accurate translations.
  • Character Level Language Modeling Using GPT: This Code Post focuses on implementing a character-level language model using the Generative Pre-trained Transformer (GPT) architecture. The objective is to generate coherent text by predicting the next character in a sequence based on previous characters, leveraging the powerful capabilities of GPT.
